# West Side Runner 🏃

Daily run forecast for the West Side Highway, NYC.
Optimized for 96th St — scores every hour 5am–9pm on temp, rain, wind,
humidity, and UV. Recommends the best 2-hour window, which direction to
start, and what to wear.

## Local development

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to vercel.com → New Project → Import your repo
3. Vercel auto-detects Vite — just click Deploy
4. Done. Your app is live.

## Connect a custom domain

1. Buy a domain (Namecheap, Google Domains, etc.)
2. In Vercel: Project Settings → Domains → Add your domain
3. Follow Vercel's DNS instructions (takes ~10 min to propagate)
